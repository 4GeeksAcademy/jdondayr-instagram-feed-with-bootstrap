>[!NOTE]
Este es el segundo proyecto que realizo durante el Bootcamp de Desarrollo Web Full-Stack de 4Geeks Academy.

He realizado, usando Bootstrap, un layout, que muestra imágenes en fila, y también en cuadrícula, así como un menú desplegable de opciones y una funcionalidad para añadir publicaciones.

Las tecnologías que he utilizado para el proyecto son:
- HTML5
- CSS
- Framework Bootstrap

Con este proyecto, nos pretendían enseñar que podemos apoyarnos en Frameworks para hacer nuestro trabajo de forma más eficiente, y en algunos casos, rápida, pero también que es importante saber como funcionan las cosas por debajo, y que nunca podemos descuidar eso.
